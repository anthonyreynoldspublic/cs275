Reflection (sample ≤ 250 words)

Which variables did you name first, and why? Colors and spacing, because they drive the biggest visible deltas and reduce “magic numbers.”

What rules can be enforced? Consistent spacing via --space-*, semantic color use via --color-primary on interactive elements, and clear type hierarchy with --fs-*.

What did the LLM suggest that you changed? I replaced a low-contrast dark palette with more legible values.

This demo uses only HTML + CSS. Dark mode uses @media (prefers-color-scheme: dark).
