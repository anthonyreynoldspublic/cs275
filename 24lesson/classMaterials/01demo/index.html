Counter Factory

Each call to makeCounter() stamps out a fresh scope instance with its own count binding. The two counters below share no state.

the factory
function makeCounter() {
  let count = 0;        // born in this scope instance

  return function increment() {
    count++;           // closes over ↑ that specific count
    return count;
  };
}

const a = makeCounter();  // scope A — count starts at 0
const b = makeCounter();  // scope B — count starts at 0 (independent)
a scope A
count = 0
b scope B
count = 0
return value log
— call a() or b() to begin —
what to notice
  • Incrementing a never changes b's count — they own separate heap-promoted scopes.
  • makeCounter is a template. Each invocation is a distinct scope instance.
  • count is unreachable from outside — no property, no global. Only the returned function can touch it.
